# IDragHandler

모바일용 게임에 사용하는 조이스틱을 만들기 위해서 **Drag** 기능을 사용하였습니다.
이 때, **IDragHandler IBeginDragHandler, IEndDragHandler** 인터페이스를 사용합니다.

- 인터페이스들을 사용하기 위해서 **using UnityEngine.Eventhandler** 을 추가합니다.
```csharp
using UnityEngine;
using UnityEngine.EventSystems;
```
